# Flexiconv documentation

Full user documentation lives in this `wiki/` folder as Markdown files. Use this index to find the right page.

| Page | Description |
| --- | --- |
| [Home](Home.md) | Overview, CLI usage, key features, and links to other pages |
| [Installation](Installation.md) | Editable install, wrapper script, optional extras (RTF, TEI-CORPO, Annatto) |
| [QuickStart](QuickStart.md) | Common `flexiconv` command patterns and recipes |
| [Deduplication](Deduplication.md) | Exact and near-identical duplicate detection, index (SQLite), incremental and cross-format |
| [Formats](Formats.md) | Overview of all supported formats and links to per-format pages |

Start with [Home](Home.md) for the full overview, or jump directly to [Formats](Formats.md) to see what Flexiconv can read and write.

**Example files.** The repository’s [`examples/`](../examples/) folder contains sample files for many of the supported formats. These are typically taken from real corpus or tool outputs and can be used to try conversions (e.g. `flexiconv examples/trs/sample.trs out.xml`).
